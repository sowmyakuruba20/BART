This folder houses a straightforward Flask application that establishes a connection with the Neo4j backend. It enables the execution of time and fare queries on the graph created in Neo4j.

To utilize the application, simply run the app.py file. The application provides an intuitive interface to select both the source and destination stations. By leveraging the power of the "shortest path" algorithm, it accurately determines the time and fare required to reach the destination station.

Whether you're a developer exploring the codebase or a user looking to calculate BART travel information, this application simplifies the process. It offers a user-friendly way to select stations, obtain relevant data, and make informed decisions about your BART journey.
